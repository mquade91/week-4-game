Crystal Collection Game

Directions:


A random number will generate at the start of the game


There are four crystals below. Each crystal is worth an unkown amount and each time you click one, the value of that crystal will be added to your total score. *Crystals can have the same value The value of the Crystal will not change until the game ends.


In order to WIN the game, your total score must equal the number given to you by the computer.


The value of each crystal, along with the number you are trying to match will reset each time you WIN or LOSE.
